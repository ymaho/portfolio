Pretty Flappy Birdy

Not just a flappy bird game: the aesthetic designed Flappy Bird game with dynamic leaderboard and scoring system.

The rule is simple: press the space button to help the bird to survive longer than other players to win. Crashing into objects or falling on the ground will be game over. Since the game is full-stuck developed, users are allowed to see othersâ€™ real-time score which makes them more competitive.

The aesthetic design is to increase their enjoyment as well as for marketing purpose. We believe the users are more likely to share the experience on social media platforms when it is "intagramable". When the users do so, the game would have free-advertisements o large range of user types, and potentially increase user amount.

Reddit Text Analysis

A program that allows us to conduct text analysis
with visualization technique on a chat group in Reddit.

Started the project with handling the Reddit authentication system with a python library called praw. After that, I did data cleaning such as removing emoji(includes symbols and pictographs), non english/numeric characters, and stopwords.

Linking the program to a generating wordcloud website is to provide a wordcloud image. The image visualizes radius of the trending topic or important words by differenciate word sizes which makes it easier for the users to understand what's going on within the group chat without scrolling all the way.
