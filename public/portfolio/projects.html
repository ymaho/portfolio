Pretty Flappy Birdy

Not just a flappy bird game: the aesthetic designed Flappy Bird game with dynamic leaderboard and scoring system.

The rule is simple: press the space button to help the bird to survive longer than other players to win. Crashing into objects or falling on the ground will be game over. Users are allowed to see othersâ€™ real-time score which makes them more competitive.


The goal of the aesthetic design is to increase their enjoyment as well as for marketing purpose. I believe that the users are more likely to share the experience on social media platforms when it is "instagramable". When the users do so, the game would have free-advertisements to various types of users, and potentially increase user amount.
Used: HTML, CSS, JavaScript, Express.js, Adobe Illustrator, Git/GitHub

Reddit Text Analysis

A program that allows us to perform sentiment analysis in a subreddit with visualization technique.

Started the project with handling the Reddit authentication system with a python library called praw. After that, I did data cleaning such as removing emoji(includes symbols and pictographs), non english/numeric characters, and stopwords.


Linking the program to a generating wordcloud website is to provide a wordcloud image. The image visualizes trending topic and important words by differenciating word sizes which makes it easier for the users to understand what's going on within the group chat without scrolling all the way.
Used: Python, praw, pandas, re, nltk, Git/GitHub
